# 財表理論アプリ

財務諸表論の理論を「読む」だけで終わらせず、答案として再現するための学習アプリです。

## 現在の初回版

- GitHub Pagesで公開できる静的SPA
- 教材カード367件を `data/cards.seed.json` から読み込み
- 1日の出題数は一旦12問
- 学習履歴、答案、自己評価、次回復習日はブラウザのlocalStorageに保存
- ホーム、今日やること、問題演習、解答確認、復習予定、学習分析、設定を実装
- 学習モードとして、章集中、要点確認、キーワード暗記、答案演習、白紙再現、弱点特訓、横断復習、過去・直前、本番タイマーを実装

## ローカル確認

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` を開きます。

## GitHub Pages

`main` ブランチへpushすると、`.github/workflows/pages.yml` により静的ファイルがGitHub Pagesへデプロイされます。
